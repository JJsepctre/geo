# 操作演示视频配置说明

## 概述
首页的操作展示模块现在支持为每个选项卡显示不同的视频内容。每个选项卡（地震波反射、电磁波反射、掌子面素描、加深炮孔、超前水平钻）都有独立的内容描述和演示视频。

## 视频文件位置

视频文件应放置在 `public/videos/` 目录中。如果该目录不存在，请手动创建。

## 所需视频文件

请准备以下视频文件并放置到 `public/videos/` 目录：

### 必需的视频文件：

1. **地震波反射演示视频**
   - `地震波反射-demo.mp4` 或 `地震波反射-demo.webm`
   - `地震波反射-poster.jpg` (视频封面图，可选)

2. **电磁波反射演示视频**
   - `电磁波反射-demo.mp4` 或 `电磁波反射-demo.webm`
   - `电磁波反射-poster.jpg` (视频封面图，可选)

3. **掌子面素描演示视频**
   - `掌子面素描-demo.mp4` 或 `掌子面素描-demo.webm`
   - `掌子面素描-poster.jpg` (视频封面图，可选)

4. **加深炮孔演示视频**
   - `加深炮孔-demo.mp4` 或 `加深炮孔-demo.webm`
   - `加深炮孔-poster.jpg` (视频封面图，可选)

5. **超前水平钻演示视频**
   - `超前水平钻-demo.mp4` 或 `超前水平钻-demo.webm`
   - `超前水平钻-poster.jpg` (视频封面图，可选)

## 视频格式建议

- **格式**: MP4 (H.264编码) 或 WebM
- **分辨率**: 建议 1280x720 或 1920x1080
- **文件大小**: 建议小于 50MB 以确保快速加载
- **封面图**: JPEG 格式，建议 1280x720

## 注意事项

1. 视频文件名必须严格按照上述格式命名（包括中文字符）
2. 如果视频文件不存在，播放器会显示"您的浏览器不支持视频播放"
3. 封面图（poster）是可选的，如果不存在也不会影响视频播放
4. 建议同时提供 MP4 和 WebM 格式以确保浏览器兼容性

## 修改内容总结

✅ 每个选项卡的卡片内容现在与标题一致
- 地震波反射选项卡：所有描述都针对地震波反射
- 电磁波反射选项卡：所有描述都针对电磁波反射
- 掌子面素描选项卡：所有描述都针对掌子面素描
- 加深炮孔选项卡：所有描述都针对加深炮孔
- 超前水平钻选项卡：所有描述都针对超前水平钻

✅ 每个选项卡的视频是独立的
- 视频标题显示对应的类型
- 视频源文件根据类型动态加载
- 切换选项卡时视频会重新加载对应的内容

## 测试建议

1. 切换到每个选项卡，检查文字描述是否正确对应
2. 准备好视频文件后，验证每个选项卡的视频是否能正常播放
3. 确认视频封面图显示正常
